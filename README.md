# Fake News Detection using Machine Learning

An AI-powered web application that detects whether a news article is Real or Fake using Natural Language Processing and Machine Learning.

## Project Overview

This project is a Fake News Detection System built using:

 Python

 Machine Learning (Logistic Regression)

 Natural Language Processing (TF-IDF Vectorization)

 Streamlit (for Web Interface)

 ## Machine Learning Workflow

Data Collection

Data Preprocessing

Removing stopwords

Lowercasing

Cleaning punctuation

Text Vectorization using TF-IDF

Model Training using Logistic Regression

Model Evaluation

Deployment using Streamlit
